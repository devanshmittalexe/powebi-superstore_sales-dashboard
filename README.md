# 📊 Power BI Sales Dashboard

## 🚀 Overview
This is my first Power BI dashboard where I analyzed sales data and built interactive visualizations.

## 📊 Features
- Total Sales KPI
- Sales Trend over Time
- Sales by Region & Category
- Interactive Slicer (Region filter)
- Sales Contribution % using DAX

## 🧠 What I Learned
- Creating dashboards in Power BI
- Writing DAX measures (SUM, CALCULATE, ALL, DIVIDE)
- Understanding filter context
- Building interactive reports

## 📁 Files
- Power BI file (.pbix)
- Dataset (.csv)
